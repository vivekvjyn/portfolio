## Use template

1. **Clone the Repository:**
```bash
git clone https://codeberg.org/vivekvjyn/vivekvjyn.git
cd vivekvjyn
```
2. **Build the website:**
```bash
npm install
npm install tailwindcss @tailwindcss/postcss postcss
# make changes
npm run build
```
